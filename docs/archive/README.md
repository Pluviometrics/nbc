# Archive

Historical session notes preserved for traceability. New work should not reference these directly.

Contents:

- `superpowers/specs/` — original feature specs from earlier work sessions
- `superpowers/plans/` — implementation plans matching the specs above

These remain in the repo so commit messages and PR descriptions that cite them stay resolvable, but they reflect old design decisions and have been superseded by the live code in `index.html`. For current behaviour see `RUNBOOK.md` and `docs/inputs.md`.
